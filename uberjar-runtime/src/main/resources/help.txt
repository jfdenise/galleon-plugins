Uberjar
Usage: java [uberjar system properties] -jar <uberjar filename>.jar [arguments]

Where arguments includes:
    -b=<value>                          Set system property jboss.bind.address 
                                        to the given value

    --cli-script=<script>               CLI script to execute at server launch. Alternative
                                        is wildfly.uberjar.cli.script system property.

    -D<name>[=<value>]                  Set a system property.

    -h, --help                          Display this message and exit.

    --no-delete                         Do not delete the server directory when exiting.

    --server-config=<config>            An external XML server configuration file path
                                        to be used by the server. Alternative is 
                                        wildfly.uberjar.server.config system property.

    --server-dir=<dir>                  Directory in which the server is installed. 
                                        By default a tmp directory is created.

    -u=<value>                          Set system property 
                                        jboss.default.multicast.address to the 
                                        given value.

    -v, --version                       Print version and exit.
